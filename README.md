# 🍔 Food In - Restaurant Website

A modern and responsive restaurant website built using HTML, CSS, and JavaScript. The project showcases a professional food ordering and restaurant landing page with attractive UI design, service highlights, customer testimonials, and mobile-friendly responsiveness.

## 🚀 Live Demo

👉 https://fastfoodsrestaurant.netlify.app/

![Restaurant Preview](restaurant-home.png)

## 📌 Features

- Responsive Design
- Modern Restaurant UI
- Interactive Navigation
- Food Services Section
- Customer Testimonials
- About Us Section
- Mobile-Friendly Layout
- Fast Loading Design
- Cross-Browser Compatibility

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)

```bash
index.html
```

## 💻 Installation

```bash
git clone https://github.com/devanshkolhe14/restaurant.website.git

cd restaurant.website
```

Open `index.html` in your browser.
